👁 Therapheye

Política de Privacidad

Última actualización: 26 de marzo de 2026

Therapheye ("nosotros", "nuestro") es una aplicación web y extensión de navegador diseñada para ayudar a los usuarios a proteger su salud visual mediante el seguimiento del tiempo de pantalla y recordatorios de descanso. Esta política explica qué datos recopilamos, cómo los usamos y cómo los protegemos.

1. Datos que recopilamos

Recopilamos únicamente la información necesaria para el funcionamiento de la aplicación:

  • Información de cuenta: nombre, correo electrónico y contraseña (almacenada como hash seguro) cuando te registras.
  • Datos del temporizador: tiempo acumulado de uso de pantalla, estado del temporizador (activo/pausado), marcas de tiempo de inicio y pausa.
  • Historial de actividad: registro de sesiones de ejercicios visuales completados, resultados de evaluaciones de salud visual y cuestionarios.
  • Datos de navegación (extensión): la extensión registra el tiempo que pasas navegando para calcular tu tiempo de pantalla. No registramos las URLs específicas que visitas ni el contenido de las páginas.

2. Cómo usamos tus datos

Utilizamos la información recopilada exclusivamente para:

  • Autenticar tu cuenta y mantener tu sesión activa.
  • Rastrear tu tiempo de pantalla y enviarte recordatorios de descanso visual.
  • Sincronizar el estado del temporizador entre la aplicación web y la extensión del navegador.
  • Mostrar tu historial de ejercicios, evaluaciones y progreso.

3. Almacenamiento y seguridad

Tus datos se almacenan en una base de datos segura en Neon (PostgreSQL serverless) con conexiones cifradas mediante TLS/SSL. Las contraseñas se almacenan como hashes criptográficos y nunca en texto plano. Los datos del temporizador se almacenan localmente en tu navegador (localStorage) y se sincronizan con la base de datos.

4. Compartición de datos con terceros

No vendemos, alquilamos ni compartimos tus datos personales con terceros. No utilizamos tus datos con fines publicitarios. No transferimos datos a terceros excepto los servicios esenciales para el funcionamiento de la aplicación (hosting en Netlify, base de datos en Neon).

5. Permisos de la extensión

La extensión de navegador solicita los siguientes permisos:

  • storage: para guardar localmente la configuración y el estado del temporizador.
  • alarms: para programar recordatorios de descanso visual.
  • tabs: para abrir la aplicación web desde la extensión.
  • host_permissions: para comunicarse con la base de datos y sincronizar tu temporizador.

6. Retención y eliminación de datos

Conservamos tus datos mientras mantengas una cuenta activa. Puedes solicitar la eliminación completa de tu cuenta y todos los datos asociados contactándonos por correo electrónico. Los datos del temporizador del día se reinician automáticamente cada día.

7. Menores de edad

Therapheye no está dirigido a menores de 13 años. No recopilamos intencionalmente información de menores de 13 años.

8. Cambios a esta política

Podemos actualizar esta política ocasionalmente. Publicaremos cualquier cambio en esta página con la fecha de actualización correspondiente.

9. Contacto

Si tienes preguntas sobre esta política de privacidad o sobre tus datos, contáctanos en: therapheye@gmail.com
